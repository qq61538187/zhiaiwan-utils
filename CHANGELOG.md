# Changelog

All notable changes to this project are tracked via [Changesets](https://github.com/changesets/changesets).

## How Release Notes Are Produced

- Add a changeset in pull requests that affect users.
- Merge to `main` to let the release workflow create/update version PRs.
- After version PR merge, publish is automated and release notes are reflected in git tags and package metadata.

## Current Series

- `0.1.x`: initial public baseline for utility APIs, ESM/CJS outputs, and typed entrypoints.
