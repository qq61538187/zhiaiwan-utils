# Math

## Methods

- [`.add`](#add)

### add(a, b)
<a id="add"></a>

Adds two numbers.

#### Since

`+0.1.0`

#### Arguments

1. `a` *(number)*: The first addend.
2. `b` *(number)*: The second addend.

#### Returns

*(number)*: Returns the arithmetic sum.

#### Example

```ts
import { add } from '@zhiaiwan/utils'

add(1, 2)
// => 3
```
