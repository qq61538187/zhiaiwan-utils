# API Categories

`@zhiaiwan/utils` API is organized by categories, similar to Lodash docs navigation.

## Categories

- [Array](/api/array)
- [Function](/api/function)
- [Object](/api/object)
- [Math](/api/math)
- [Grouped Exports](/api/grouped)
- [Full Entry](/api/full-entry)

Each category page includes:

- A list of methods in that category.
- Per-method sections with `Since`, `Arguments`, `Returns`, and `Example`.
- Sidebar anchors so clicking a method jumps to the corresponding section.
